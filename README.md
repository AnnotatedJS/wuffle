# Wuffle Board

A task board for [GitHub](https://github.com) issues.

![Wuffle Screenshot](./docs/screenshot.png)


## Try it

```
npm install
npm start
```

Goto [`localhost:3001/board`](http://localhost:3001/board) to inspect the board.
